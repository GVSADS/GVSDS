## 📋 关于我们
银河万通软件开发工作室（Galaxy Universal, Inc.）是一支专注于软件研发、技术咨询、数字化解决方案的专业IT团队，致力于为客户提供高效、可靠、可扩展的技术产品与服务。

### 核心能力
- ✅ 全栈软件开发（前端/后端/移动端）
- ✅ 定制化系统解决方案
- ✅ 技术架构设计与优化
- ✅ 代码重构与性能调优
- ✅ 开源技术生态整合

### 技术栈
| 领域       | 核心技术                                                                  |
|------------|--------------------------------------------------------------------------|
| 前端       | HTML5/CSS3/JavaScript、React、TailwindCSS                                 |
| 后端       | Java、Python、Go、Node.js、Spring Boot、flask、Django、微服务架构           |
| 移动端     | 小程序、React、Flutter                                                     |
| 数据库     | MySQL、MongoDB、Redis                                                     |
| 部署/运维  | Docker、K8s、CI/CD、Nginx、云服务（阿里云）                                 |
